# 🍔 Food Ordering Website

A simple and responsive Food Ordering Website built using HTML, CSS, and JavaScript. This project allows users to view food items and place orders through an interactive interface.

## 🚀 Features

- View food items and prices
- Order food with a single click
- Interactive JavaScript functionality
- Responsive and user-friendly design
- Clean and simple interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

Food-ordering/
│
├── index.html
├── style.css
├── script.js
└── README.md

## ▶️ How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Double-click `index.html` or run it using Live Server in VS Code.
